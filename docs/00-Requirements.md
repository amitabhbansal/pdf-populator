**Activity: Populate Data into Scanned PDFs Without Using LLMs**

**Objective**

Design and implement a solution to automatically populate values into a scanned PDF document that contains no editable or searchable form fields.

**Constraints**

- Input PDF is a scanned document (image-based PDF).
- No existing AcroForm/XFA fields.
- No Large Language Models (LLMs) may be used.
- Solution should be automatable.
- Accuracy and repeatability are important.

**Deliverables**

1. Solution design document.
2. Architecture diagram.
3. Working prototype.
4. Sample input and output PDFs.
5. Accuracy and performance metrics.
6. Deployment considerations.
